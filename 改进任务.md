# LookAway 代码改进与优化任务清单

这是一个关于 LookAway 代码重构与优化的待办任务清单。我们可以在接下来的工作中逐项修改并验证。

## 🔴 1. 潜在 Bug (高优先级)

- [x] **RestSession onComplete 闭包循环引用与并发警告**
  - **描述**：`RestSession` 初始化时传入的 `onComplete` 闭包需要增加 `@Sendable @MainActor` 属性，以满足 Swift 5.9+ Strict Concurrency 线程安全检查，并规范主线程调用。
  - **文件**：[RestSession.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/Services/RestSession.swift) / [AppDelegate+RestWindow.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/AppDelegate+RestWindow.swift)
- [x] **closeRestWindow 中挂起状态的不一致性**
  - **描述**：系统挂起时，直接在 `closeRestWindow` 中将 `isSystemSuspended` 强制置为 `true` 会导致状态机不一致。应调用统一的 `suspendWorkCountdownForInactiveSystem()` 函数。
  - **文件**：[AppDelegate+RestWindow.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/AppDelegate+RestWindow.swift)
- [x] **计时器防作弊/防误触（锁屏与睡眠时长校验）**
  - **描述**：如果系统挂起时间小于设定的休息时间（如误触锁屏 2 秒即解锁），应恢复之前的倒计时而非直接重置为满格。
  - **文件**：[AppDelegate+SystemActivity.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/AppDelegate+SystemActivity.swift)
- [x] **多屏幕配置热插拔适配**
  - **描述**：监听 `NSApplication.didChangeScreenParametersNotification`，若在休息期间屏幕发生插拔，自动重新绘制/调整遮挡窗口，防止新接屏幕被绕过。
  - **文件**：[AppDelegate+SystemActivity.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/AppDelegate+SystemActivity.swift)

  - **文件**：[VideoPauser.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/Services/VideoPauser.swift)

  - **文件**：[AppDelegate.swift](file:///Users/0x727a/编程/lookaway/Sources/lookaway/AppDelegate.swift)
